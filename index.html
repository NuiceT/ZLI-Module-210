Public Cloud

Was weiss ich über Public Cloud?

Server mieten -> Meistens pro Stunde

Wenn man einen grossen Prozess starten muss, aber nur kurz, dann sind grosse Server nützlich. Das Problem ist aber, man will jetzt nicht extra einen mieten, weil das mit vielen kosten verbunden ist. In der Public Cloud kann man Server pro Stunde mieten, das ist somit günstiger und schneller.

Man kann automatisch skalieren. Das heisst: Wenn aus 100 Besuchern pro Stunde plötzlich 10'000 werden, dann kann man (automatisiert) mehr Cloud Services/Container starten mit einem Load Balancer, um den Traffic zu überwinden.
